Briefkasten comes with a REST API reachable at `https://briefkastenhq.com/api/v1/**`. These are based on [SvelteKit `+server` Routes](https://kit.svelte.dev/docs/routing#server) and the code can be found in the [monorepo](https://github.com/ndom91/sveltekasten) at `/apps/web/src/routes/api/v1/*`.

## Authentication

You can get your API token from the [Settings page](/settings).

import APIToken from "../../public/screenshots/api-token.png"

<Screenshot src={APIToken} alt="Settings -> API Token" className="shadow-xs" />

Then you can use your token to authenticate by passing it in the `Authorization` Header.

```ts title="createBookmark.ts"
const response = await fetch("https://briefkastenhq.com/api/v1/bookmarks", {
  method: "POST"
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer <TOKEN>", // [!code highlight]
  },
  body: JSON.stringify({
    url: "https://reddit.com",
    title: "Reddit",
  })
})

const body = await response.json()
```
